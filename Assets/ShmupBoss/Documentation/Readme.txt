If you notice a dark demo scene lihgting. Go to "Window" --> "Rendering" --> "Lighting" then press "Generate Lighting" button.

This project is using the built-in renderer to make it easy to upgrade into either URP or HDRP.

To upgrade into URP: https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@7.1/manual/InstallURPIntoAProject.html

To upgrade into HDRP: https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@14.0/manual/Upgrading-To-HDRP.html
